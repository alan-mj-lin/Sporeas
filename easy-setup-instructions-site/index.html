Sporeas - Easy Setup Instructions

  1. If it's your first time here, the file should already be automatically downloading to the Sporeas folder. After a few minutes, the Python installer should then automatically start. If it doesn't, you might need to download it manually (click here).
  2. When the Python installer is downloaded and running, simply follow the instructions.
  3. When the Python installer's done, run the rest of the setup
  4. http://127.0.0.1:9000/admin will automatically open. Wait about 5 seconds and refresh the page.
  5. If you want to stop Sporeas (it's running in the background), then on the "stop" file
